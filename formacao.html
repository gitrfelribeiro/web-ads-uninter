Formação

Atualmente, curso Análise e Desenvolvimento de Sistemas pela UNINTER, uma formação voltada para o desenvolvimento de soluções computacionais, com foco em programação, banco de dados, engenharia de software, arquitetura de sistemas, segurança da informação e infraestrutura tecnológica.

Ao longo do curso, tenho adquirido uma base sólida em lógica de programação, estruturas de dados, modelagem de sistemas, desenvolvimento web, testes de software e projetos orientados a objetos.

Paralelamente aos conteúdos acadêmicos, tenho direcionado meus estudos de forma estratégica para a área de dados, com o objetivo de me tornar um engenheiro de dados altamente qualificado.

Tenho aprofundado meus conhecimentos em processos de ETL, bancos de dados relacionais e não relacionais, visualização de dados com Power BI, tratamento e transformação de dados, além de ferramentas e práticas modernas como computação em nuvem, containers (Docker), CI/CD, Data Warehousing e arquitetura de dados distribuída.

Todo esse investimento tem como propósito aperfeiçoar minha carreira profissional na área de dados, contribuindo para que eu ofereça soluções inteligentes e orientadas à realidade dos negócios.
